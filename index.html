    Pushup Ninja

    Web App Developer

    Push-Up Ninja is a web app that challenges users to test themselves with push-up exercises and improve their score, all while it tracks their stats and provides motivational and practical feedback. The app uses the Myo Armband peripheral, a wearable gadget that reads electrical impulses from the muscles of the forearm. The device uses these to detect when the user is engaged in a proper push-up, and sends a signal to increment the counter on the website.

    Jungle Run

    Game Designer & Programmer
